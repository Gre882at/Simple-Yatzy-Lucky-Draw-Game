# Simple-Yatzy-Lucky-Draw-Game
This is a beginner-friendly Java program that collects user input in both string and integer form to play two simple games: Yatzy and Lucky Draw.
#Features
-Accepts user input using the Scanner class.
-uses string and integer variables to handle input.
-Includes two easy-to-understand and fun games.
